# notes-web-app

Note-taking web app.

Tech stack: Next.js, React, TailwindCSS.

## Setup

1. Install the dependencies:
    ```bash
    npm install
    ```

2. Set env variables
    ```bash
    export token=<token>
    ```

3. Run the development server:
    ```bash
    npm run dev

4. Open [http://localhost:3000](http://localhost:3000) to see the result.
